# Joining as a member

> [!NOTE]
> This is for individuals seeking to register themselves as a member.

Create a new TOML file in `members/` with your **GitHub username** as the filename:

```toml
full-name = "Your Name"
andrew-id = "your-andrew-id"
```

Visit the [member schema](../../meta/schemas/member.schema.json)
to learn more about each field.
